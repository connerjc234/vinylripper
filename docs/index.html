Features

Everything you need to digitize vinyl

A focused toolkit that turns your turntable into a high-quality digital recording studio.

Live Recording

Record directly from your sound card with real-time waveform display. Supports any input device at any sample rate.

Auto Silence Detection

Automatically detects gaps between tracks using FFmpeg. Adjustable sensitivity and minimum gap duration for perfect splits every time.

Discogs Integration

Look up album metadata from Discogs — artist, title, year, label, tracklist, and cover art. Tags your files automatically.

Multi-Format Export

Save as FLAC (0-8 compression), MP3 (VBR 0-9), or AIFF (16/24/32-bit). Embedded metadata and cover art included.

Live Waveform + Undo/Redo

Watch your recording in real time. Review with draggable split markers, undo/redo (Ctrl+Z/Ctrl+Y), and minimap overview.

Vinyl Track Numbering

Automatic A1/A2/B1/B2 positions from Discogs data. Sub-track/medley support (A1.a, A1.b joined titles).

Audio Restoration

Optional highpass rumble filter (30Hz) and declicking via FFmpeg. Clean up your recordings before export.

Persistent Settings

JSON config with environment variable overrides. Settings dialog for format, quality, restoration level, and Discogs token.

How It Works

From vinyl to files in four steps

The workflow is straightforward. Connect your turntable, search for your album, record, and save.

1

Connect & Configure

Plug your USB turntable (or a standard turntable via a USB audio interface) into your computer. Select the input device from the dropdown.

2

Search Discogs

When you press Record, a search dialog opens. Search for your album on Discogs by artist and title. Select the matching release to pull in track names, year, and cover art.

3

Record Your Vinyl

Hit Record and start playing your record. The waveform updates in real time. When the side is done, press Stop.

4

Split & Save

Review the waveform, adjust split markers, and save individual tracks as FLAC files with full metadata and cover art embedded.

Installation

Get up and running

Requires a USB turntable or audio interface, Python 3.11 or later. Supports Linux, macOS, and Windows.

# Debian / Ubuntu / Mint — install system deps sudo apt install ffmpeg portaudio19-dev libxcb-cursor0 # Clone the repository git clone https://github.com/connerjc234/vinylripper.git cd vinylripper # Create a virtual environment python3 -m venv .venv source .venv/bin/activate # Install Python dependencies pip install -e . # Run vinylripper
# Fedora / RHEL sudo dnf install ffmpeg portaudio-devel git clone https://github.com/connerjc234/vinylripper.git cd vinylripper && python3 -m venv .venv && source .venv/bin/activate pip install -e . vinylripper
# Arch Linux sudo pacman -S ffmpeg portaudio git clone https://github.com/connerjc234/vinylripper.git cd vinylripper && python3 -m venv .venv && source .venv/bin/activate pip install -e . vinylripper

Or run bash setup.sh for the quick-start script.

# Install FFmpeg + PortAudio via Homebrew brew install ffmpeg portaudio # Clone the repository git clone https://github.com/connerjc234/vinylripper.git cd vinylripper # Create a virtual environment python3 -m venv .venv source .venv/bin/activate # Install Python dependencies pip install -e . # Run vinylripper

Tested on Apple Silicon (M1/M2/M3) and Intel macOS.

# Install FFmpeg from https://ffmpeg.org/download.html and add to PATH # Install PortAudio via conda-forge conda create -n vinylripper python=3.11 conda activate vinylripper conda install -c conda-forge portaudio # Clone the repository git clone https://github.com/connerjc234/vinylripper.git cd vinylripper # Create a virtual environment python -m venv .venv .venv\Scripts\activate # Install Python dependencies pip install -e . # Run vinylripper

⚠ Windows is experimental. Requires manual FFmpeg install and conda for PortAudio.

You'll need a Discogs personal access token for metadata lookup.

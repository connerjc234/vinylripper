Features

Everything you need to digitize vinyl

A focused toolkit that turns your turntable into a high-quality digital recording studio.

Live Recording

Record directly from your sound card with real-time waveform display. Supports any input device at any sample rate.

Auto Silence Detection

Automatically detects gaps between tracks. Adjustable sensitivity and minimum gap duration for perfect splits every time.

Discogs Integration

Look up album metadata from Discogs — artist, title, year, label, tracklist, and cover art. Tags your FLAC files automatically.

FLAC with Cover Art

Save as FLAC, WAV, OGG, or AIFF. Embedded metadata and cover art included. Your digital library stays organized.

Live Waveform

Watch your recording in real time with a smooth, scrolling waveform display. Review your waveform after recording with draggable split markers.

Track Splitting

Split your recording into individual tracks automatically. Draggable markers let you fine-tune split points. Name tracks from Discogs data.

How It Works

From vinyl to files in four steps

The workflow is straightforward. Connect your turntable, search for your album, record, and save.

1

Connect & Configure

Plug your turntable into your computer's line-in or USB audio interface. Select the input device from the dropdown.

2

Search Discogs

When you press Record, a search dialog opens. Search for your album on Discogs by artist and title. Select the matching release to pull in track names, year, and cover art.

3

Record Your Vinyl

Hit Record and start playing your record. The waveform updates in real time. When the side is done, press Stop.

4

Split & Save

Review the waveform, adjust split markers, and save individual tracks as FLAC files with full metadata and cover art embedded.

Installation

Get up and running

Requires Python 3.11 or later. Supports Linux, macOS, and Windows.

# Debian / Ubuntu / Mint — install system deps sudo apt install portaudio19-dev libxcb-cursor0 # Clone the repository git clone https://github.com/connerjc234/vinylripper.git cd vinylripper # Create a virtual environment python3 -m venv .venv source .venv/bin/activate # Install Python dependencies pip install -e . # Run vinylripper
# Fedora / RHEL sudo dnf install portaudio-devel git clone https://github.com/connerjc234/vinylripper.git cd vinylripper && python3 -m venv .venv && source .venv/bin/activate pip install -e . vinylripper

Or run bash setup.sh for the quick-start script.

# Install PortAudio via Homebrew brew install portaudio # Clone the repository git clone https://github.com/connerjc234/vinylripper.git cd vinylripper # Create a virtual environment python3 -m venv .venv source .venv/bin/activate # Install Python dependencies (macOS reqs include PyQt6 via pip) pip install -e . # Run vinylripper

⚠ macOS is untested. If you hit issues, open a GitHub issue.

# Install PortAudio via conda-forge conda create -n vinylripper python=3.11 conda activate vinylripper conda install -c conda-forge portaudio # Clone the repository git clone https://github.com/connerjc234/vinylripper.git cd vinylripper # Create a virtual environment python -m venv .venv .venv\Scripts\activate # Install Python dependencies pip install -e . # Run vinylripper

⚠ Windows is untested. Requires conda or a manual PortAudio build. Open an issue if you need help.
